# moonlight-embedded-gui

Electron GUI for [moonlight embedded](https://github.com/irtimmer/moonlight-embedded)

## Software stack

- Node
- Electron
- Vue 2.0
- Vuex

## Requirements

- Raspberry Pi
- Arch linux
- **moonlight** installed
